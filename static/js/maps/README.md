# static/js/maps/

SVG pipeline maps for the Natural Gas and Crude Oil sector pages. These render interactive maps showing major pipeline networks and trading hubs.

## Files

| File | Purpose |
|------|---------|
| `maps.js` | Map rendering engine — draws SVG maps, hub markers, tooltips, click-to-trade from map locations |
| `pipeline-network.js` | Pipeline route coordinates and styling for major US pipeline systems |
| `world-paths.js` | Country border SVG paths (Mercator projection) for the world map background |
| `state-paths.js` | US state border SVG paths for the domestic pipeline map |
